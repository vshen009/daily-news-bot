# Claude 长指令模板

## 项目上下文

你正在协助开发 Daily News Bot 项目。

### 项目特点
- 所有者：Vincent（非程序员）
- 语言偏好：简体中文
- 数据库：SQLite（提交到 Git）
- 主分支保护：必须使用 PR
- Git 规范：emoji 前缀（✨ 新功能、🔧 修复、📝 文档等）

### 工作流程
1. 创建功能分支
2. 开发并测试
3. 提交（使用 emoji 前缀）
4. 推送并创建 PR
5. 合并 PR（squash merge）

### 核心模块
- scraper.py：抓取新闻
- database.py：数据库操作
- translator.py：翻译
- html_generator.py：生成 HTML
- main.py：主流程（9步骤）
